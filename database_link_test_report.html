Отчет о проверке связей выгрузок с клиентами и проектами

Дата проверки: 2025-11-16

Проверка структуры БД

✅ Колонки для связей существуют

В таблице uploads присутствуют следующие колонки:

  • database_id - INTEGER
  • client_id - INTEGER
  • project_id - INTEGER

Также созданы индексы для оптимизации запросов:

  • idx_uploads_database_id
  • idx_uploads_client_id
  • idx_uploads_project_id

Механизм связывания

Как работает связывание

  1. При handshake с database_id:
    • API получает database_id из XML запроса
    • Преобразует строку в integer
    • Сохраняет database_id в таблице uploads
    • Получает информацию о проекте и клиенте из service.db через project_databases
    • Обновляет client_id и project_id в таблице uploads
  2. При автоматической идентификации:
    • API ищет похожую выгрузку по параметрам (computer_name, user_name, config_name, version_1c)
    • Если найдена, использует её database_id
    • Копирует client_id и project_id из похожей выгрузки

Проверка сохранения данных

✅ Данные сохраняются корректно

Проверено:

  • Выгрузки создаются в таблице uploads
  • Колонки database_id, client_id, project_id доступны
  • Механизм обновления client_id и project_id реализован в коде

⚠️ Важные замечания

  • Миграции для полей итераций: Колонки iteration_number, iteration_label, programmer_name, upload_purpose, parent_upload_id требуют применения миграций при следующем перезапуске контейнера
  • Связи через service.db: Для получения client_id и project_id API использует таблицу project_databases в service.db
  • Кэширование: После создания выгрузки API обновляет client_id и project_id для быстрого доступа без JOIN запросов

SQL запросы для проверки связей

Получить выгрузку с информацией о клиенте и проекте:
SELECT u.id, u.upload_uuid, u.database_id, u.client_id, u.project_id,
    pd.name as db_name, cp.name as project_name, c.name as client_name
FROM uploads u
LEFT JOIN project_databases pd ON u.database_id = pd.id
LEFT JOIN client_projects cp ON pd.client_project_id = cp.id
LEFT JOIN clients c ON cp.client_id = c.id
WHERE u.id = ?;
Получить все выгрузки клиента:
SELECT u.* FROM uploads u
WHERE u.client_id = ?
ORDER BY u.started_at DESC;
Получить все выгрузки проекта:
SELECT u.* FROM uploads u
WHERE u.project_id = ?
ORDER BY u.started_at DESC;
