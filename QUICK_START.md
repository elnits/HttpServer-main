# Быстрый старт

## 1. Перезапуск Backend

**ВАЖНО**: После любых изменений в `server/server.go` необходимо перезапустить backend!

### Windows:
```bash
# Остановите текущий процесс (Ctrl+C в терминале)
# Затем запустите:
start-backend.bat
```

### Или вручную:
```bash
go run main_no_gui.go
```

## 2. Проверка работы

После перезапуска запустите тесты:
```bash
powershell -ExecutionPolicy Bypass -File test-api.ps1
```

Все endpoints должны вернуть статус 200.

## 3. Настройка Frontend

Создайте файл `frontend/.env.local`:
```env
BACKEND_URL=http://localhost:9999
SERVICE_DB_NAME=Сервисная БД
```

## 4. Запуск Frontend

```bash
cd frontend
npm run dev
```

## 5. Проверка в браузере

Откройте http://localhost:3000 и проверьте:
- [ ] Главная страница загружается
- [ ] Индикатор БД показывает "Сервисная БД"
- [ ] Страница клиентов работает
- [ ] Можно создать клиента
- [ ] Можно создать проект
- [ ] Нормализация работает

## Проблемы?

### Backend возвращает 404
→ Перезапустите backend сервер

### Frontend не может подключиться к backend
→ Проверьте, что backend запущен на порту 9999
→ Проверьте `BACKEND_URL` в `.env.local`

### Ошибки компиляции
→ Запустите `go build ./server` для проверки ошибок
→ Проверьте, что все зависимости установлены
