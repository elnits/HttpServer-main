## Как будет выглядеть страница с word-based дубликатами

### Структура отображения

Когда в базе данных будут найдены word-based дубликаты, они будут отображаться следующим образом:

```
┌─────────────────────────────────────────────────────────────┐
│ 🟢 Группировка по словам  [75% схожесть] [3 записи]        │
│ Common words (2): ноутбук, gaming                          │
│                                                             │
│ Группа дубликатов #123                                     │
│ Создано: 15.01.2025, 14:30:00                             │
│                                    [Объединить]            │
├─────────────────────────────────────────────────────────────┤
│ ⭐ Рекомендуемая мастер-запись                             │
│ ID: #45 | Код: NB-001                                      │
│ Название: ноутбук asus gaming мощный                       │
│ Категория: Electronics                                     │
│ КПВЭД: 26.20.11                                            │
│ [90% качество] [AI]                                        │
├─────────────────────────────────────────────────────────────┤
│ Все записи в группе:                                       │
│                                                             │
│ #45 ⭐ ноутбук asus gaming мощный                          │
│     Код: NB-001 | КПВЭД: 26.20.11                          │
│     [90% качество] [AI]                                     │
│                                                             │
│ #67   ноутбук lenovo gaming                                │
│     Код: NB-002 | КПВЭД: 26.20.11                          │
│     [85% качество] [База]                                  │
│                                                             │
│ #89   ноутбук dell gaming                                  │
│     Код: NB-003 | КПВЭД: 26.20.11                          │
│     [88% качество] [База]                                  │
└─────────────────────────────────────────────────────────────┘
```

### Визуальные элементы

1. **Бейдж типа дубликата**:
   - 🟢 Зеленый бейдж "Группировка по словам" для `word_based`
   - 🔴 Красный для `exact`
   - 🟠 Оранжевый для `exact_name`
   - 🔵 Синий для `semantic`
   - 🟣 Фиолетовый для `phonetic`
   - ⚫ Серый для `mixed`

2. **Бейдж схожести**:
   - Цвет зависит от процента (95%+ красный, 90%+ оранжевый, 85%+ желтый, остальные синий)

3. **Бейдж причины**:
   - Отображает список общих слов: "Common words (2): ноутбук, gaming"

4. **Мастер-запись**:
   - Выделена желтым фоном
   - Отмечена иконкой звезды ⭐
   - Показывается в отдельном блоке сверху

## Как получить дубликаты для отображения

Для того чтобы увидеть word-based дубликаты на странице, нужно:

1. **Запустить анализ качества данных**:
   - Это автоматически создаст группы дубликатов в таблице `duplicate_groups`
   - Word-based дубликаты будут найдены автоматически при вызове `AnalyzeDuplicates()`

2. **Или создать тестовые данные**:
   - Добавить записи с общими словами в названиях
   - Запустить анализ дубликатов
