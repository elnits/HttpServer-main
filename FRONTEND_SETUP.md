# Инструкция по запуску веб-интерфейса

## Быстрый старт

### 1. Запуск бэкенда (Go)

```bash
cd E:\HttpServer
go run main.go
```

Бэкенд будет доступен на `http://localhost:9999`

### 2. Запуск фронтенда (Next.js)

```bash
cd E:\HttpServer\frontend
npm install
npm run dev
```

Фронтенд будет доступен на `http://localhost:3000`

## Использование

1. Откройте браузер и перейдите на `http://localhost:3000`
2. На главной странице нажмите "Управление нормализацией"
3. На странице нормализации нажмите "Запустить нормализацию"
4. Наблюдайте за прогрессом в реальном времени:
   - Прогресс-бар показывает общий прогресс
   - Логи отображают детальную информацию
   - Статистика обновляется автоматически

## Функции интерфейса

- **Запуск нормализации** - кнопка "Запустить нормализацию"
- **Остановка процесса** - кнопка "Остановить" (доступна во время выполнения)
- **Мониторинг прогресса** - прогресс-бар и процент выполнения
- **Логи в реальном времени** - события через Server-Sent Events
- **Статистика** - количество групп, категорий, объединенных записей

## Технические детали

- Фронтенд использует SSE для получения событий в реальном времени
- API роуты в Next.js проксируют запросы к Go бэкенду
- Все компоненты используют shadcn/ui для единообразного дизайна
- Интерфейс адаптивный и работает на разных размерах экрана
